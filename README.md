# Portfolio

A personal portfolio website featuring a unique Greek Temple design, built with modern web technologies.

## Tech Stack

- **React** - UI Library
- **TypeScript** - Static Typing
- **Vite** - Build Tool & Dev Server
- **CSS Modules** - Styling

## Project Structure

- `src/pages/` - Page components (Home, Projects, Contact, etc.)
- `src/assets/` - Static assets
- `public/` - Public files (e.g., CV.pdf)

## Getting Started

1. **Install dependencies**

   ```bash
   npm install
   ```

2. **Run the development server**

   ```bash
   npm run dev
   ```

3. **Build for production**
   ```bash
   npm run build
   ```
